ЗАДАЧА ДЛЯ REPLIT: БОТ LORAN.CYBER

---

ЧТО МЫ СТРОИМ

Telegram-бота для компьютерного клуба LORAN.CYBER в Астане. Бот — это серверная часть. Web App (интерфейс) будем делать позже отдельно.

---

ЧТО УЖЕ ГОТОВО

Этап 1-4 завершены. Бот умеет:

· Регистрировать пользователей с телефоном
· Привязывать аккаунты к клубной системе Senet
· Бронировать от 1 до 5 ПК
· Отменять бронь за 30 минут до начала
· Отправлять напоминания за час
· Отмечать не пришедших (no-show)
· Управлять очередью на занятые ПК
· Спрашивать подтверждение при освобождении ПК
· Блокировать ПК на 5 минут через Senet пока ждёт ответ

---

ЧТО НУЖНО СДЕЛАТЬ СЕЙЧАС

Этап 5: Админ-панель и правки к предыдущим этапам.

---

1. АДМИН-ПАНЕЛЬ

Вход по /admin с логином и паролем. Три попытки, потом блокировка 15 минут. Пароли хешировать через bcrypt.

Разделы:

· Дашборд — брони на сегодня, загрузка ПК, выручка, уведомления
· Брони — список с фильтрами, поиск, отмена с причиной
· Клиенты — список с поиском, карточка клиента, бан/разбан
· Запросы на привязку Senet — коды LOR-VRFY, подтвердить/отклонить
· Статистика — загрузка по дням, топ клиентов, топ ПК

Главный админ может назначать других админов. Обычный админ не может.

Все действия логируются в таблицу admin_logs.

---

2. ПРАВКИ К ЭТАПУ 2 (РЕГИСТРАЦИЯ)

Добавить поле «Пароль» при регистрации. Минимум 6 символов, буквы и цифры.

Бот создаёт пользователя в Senet через API с Application Key:

```
POST /api/users
Body: { login, password, phone }
```

Пароль в боте не хранится — только передаётся в Senet.

---

3. ПРАВКИ К ЭТАПУ 3 (ЦЕНЫ)

Убрать хардкод цен. Вместо этого:

· Бот запрашивает цены у Senet API через GET /api/pricing
· Сохраняет копию в таблицу pricing
· Раз в час обновляет копию
· При создании брони считает цену по данным из этой таблицы
· Если Senet недоступен — использует последнюю сохранённую копию

То же самое с баром.

---

4. КОНТАКТЫ КЛУБА

Добавить раздел в админ-панели где админ может менять:

· Номер телефона
· WhatsApp
· Telegram-канал
· Instagram

Сохраняется в БД. Web App будет подтягивать оттуда.
